# E-commerce-Sales-Analysis-using-Excel-Python
Data analysis of e-commerce transactions using Python (Pandas) and Excel dashboards for business insights

## Objective

To analyze e-commerce transactional data and generate actionable insights on sales performance, customer behavior, and regional trends.

## Tools Used

* Microsoft Excel (Pivot Tables, Charts, Dashboard)
* Python (Pandas for data analysis)

## Key Steps

* Cleaned and structured raw data
* Performed exploratory data analysis (EDA)
* Created KPIs like revenue and sales trends
* Built a dashboard in Excel

## Key Insights

* Identified top-performing countries contributing to revenue
* Analyzed monthly sales trends
* Found high-value customers and purchasing patterns
* Highlighted best-selling products

## Outcome

Developed a dashboard to support data-driven decision-making and improve business insights.
